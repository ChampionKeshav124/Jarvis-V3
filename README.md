# JARVIS V3 - AI Desktop Assistant (Voice Enabled)

JARVIS V3 is a desktop AI assistant built using Electron and Python, now upgraded with voice capabilities using ElevenLabs.

This version transforms the assistant from a simple interface into an interactive system that can respond with real-time speech.

## Features

- 🖥️ Electron-based desktop application  
- 🧠 AI responses powered by Gemini API  
- 🔊 Voice output using ElevenLabs  
- 🎛️ Voice toggle (ON/OFF)  
- ⚡ Fast and smooth UI interactions  
- 🧩 Modular structure for future upgrades  

## Tech Stack

- Electron (Node.js, HTML, CSS, JavaScript)  
- Python (Backend processing)  
- Gemini API (AI responses)  
- ElevenLabs API (Text-to-Speech)

## Version Progression

- V1 → Basic Python assistant  
- V2 → Desktop app (Electron UI)  
- V3 → Voice-enabled AI (current)

## Upcoming Features

- Multilingual support (V4)  
- System control (V5)  
- Memory and personalization
